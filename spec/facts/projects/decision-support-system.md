# Project: Personal AI Decision-Support System
Shay built a personal AI decision-support system that turns noisy, high-volume
market data into clear, risk-managed signals. It combines a signal engine
(several weighted technical indicators), end-to-end automation, and a built-in
risk layer - loss limits, drawdown auto-stop, and cooldowns - all operated
through a Telegram control interface. Built in Python (SQLAlchemy, PostgreSQL,
Docker), it runs in a safe demo/paper mode: a research project to explore agent
design, automation, and guardrails, not a live trading product.
